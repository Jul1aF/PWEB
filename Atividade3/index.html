XSS

O que é XSS?

Cross-Site Scripting (XSS) é uma vulnerabilidade de segurança em aplicações web que permite a injeção de scripts maliciosos em páginas visualizadas por outros usuários.

Como funciona?

O ataque de XSS acontece quando scripts maliciosos são injetados em páginas legítimas para capturar informações sensíveis, como tokens de sessão.

Como prevenir?

  • Nunca confiar nos dados enviados pelo usuário
  • Filtrar caracteres especiais.
  • Converter caracteres como < e > em entidades HTML (< e >) antes de exibir no navegador.

CSRF

O que é CSRF?

O CSRF (Cross-Site Request Forgery) induz um usuário autenticado a executar ações indesejadas em uma aplicação na qual ele está logado.

Como prevenir?

  • Implementar Tokens Anti-CSRF.
  • Utilizar cookies com atributo SameSite.
  • Exigir re-autenticação para ações críticas.
